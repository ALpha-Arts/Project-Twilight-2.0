# Production Method Icon Guide

## Production

### Bevel

- Filters -> Decor -> Add Bevel
    - Level: 2
    - Bump map only
- Filters -> Map -> Bump Map
    - Type: Spherical
    - Azimuth: 90
    - Depth: 2

### Shadow

- Filters -> Blur -> Gaussian Blur
    - Level: 2
- Duplicate three same layers

## Export

- Resize
    - Dimension: 104 * 104
    - Interpolation: Cubic
- Format: DDS
- No Compression
- Generate Mipmaps
